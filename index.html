Showcase

KUMA Suite

The project is being built with Rails and PostgreSQL, with Semantic UI CSS framework. It was created within a 2 week time period for my final project for CodeCore.

Kuma Suite is an all-in-one customer support web portal for SMB. Some of the project features include: client side ticket submission, ticket management complete with flags, status' and assigned reps, administrative dashboard, internal invoicing system to be integrated with Stripe API, and a calendar for scheduling and appointments also to be implemented.

VanTech Web

A group collaboration project during CodeCore's hackathon weekend where I took on the role of Front End Dev and built the structure on Rails and Materialize CSS.

VanTech Web is a web portal that lists top tech companies in Vancouver with features for adding Organizations and viewing Meetups.
