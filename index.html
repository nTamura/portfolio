Showcase

KUMA Suite

Currently a work in progress. The project is being built with Rails and Active Record, with Semantic UI CSS framework.

Kuma Suite is an all-in-one customer support web portal for SMB. Some of the project features include: client side ticket submission, ticket management complete with flags, status' and assigned reps, administrative dashboard, internal invoicing system to be integrated with Stripe API, and a calendar for scheduling and appointments also to be implemented.
